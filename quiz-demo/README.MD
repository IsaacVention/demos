# Starter Template

## Running the backend

Make sure you have python 3.10 installed.
1. cd into the backend `cd starter_template/backend`
2. setup your virtual environment `python -m venv venv`
3. install the requirements `pip install -r requirements.txt`
4. run the server `python -m uvicorn server:app --reload`
Your server should now be running on localhost:8000

## Running the frontend
Make sure you have npm 22 or newer installed
1. cd into the frontend `cd starter_template/frontend`
2. install the requirements `npm install`
3. run the frontend `npm run dev`
Your frontend should now be running on localhost:5173
Open it in your browser to play with the demo
